Cricket Match Winner Prediction (ML Project)
Overview:

This project is an end-to-end Machine Learning application that predicts the outcome of international cricket matches.

It includes:

Pre-Match Prediction (before match starts)
Live Match Prediction (during match with real-time inputs)

The model is deployed using Streamlit.

 Features: 
🔹 Pre-Match Prediction
Team 1 vs Team 2
Toss winner & decision
Predicts match winner before the match begins

Tech Stack:

1) Python
2) Pandas, NumPy
3) Scikit-learn
4) Streamlit (for deployment)
5) Joblib (model saving)
